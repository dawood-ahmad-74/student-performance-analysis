# student-performance-analysis
**Overview**

This project demonstrates an end-to-end machine learning workflow for analyzing student performance data and generating actionable insights using Microsoft Excel and WEKA.

The solution focuses on transforming raw educational data into meaningful insights through data preparation, exploratory analysis, feature selection, predictive modeling, clustering, and model evaluation. The objective is to identify the key factors affecting performance and support data-driven decision-making.

**Business Problem**

Educational organizations often collect large volumes of student data but struggle to identify students who may require early intervention.

This project develops a machine learning solution that:

Identifies key factors affecting student performance
Predicts student performance categories
Compares multiple machine learning models
Evaluates model performance using industry-standard metrics
Generates insights to support informed decision-making

**Technologies Used**

Microsoft Excel
WEKA
CSV Dataset
ARFF Dataset
**Project Workflow**

** Data Collection**

Imported and validated the dataset
Verified attribute consistency
Examined data quality

**✔ Data Preprocessing**

Cleaned missing values
Converted CSV to ARFF format
Applied preprocessing filters
Prepared data for machine learning

**✔ Exploratory Data Analysis**

Performed statistical analysis and created visualizations including:

Histogram Analysis
Student Distribution Analysis
Average Score Analysis
Pearson Correlation Analysis

**✔ Feature Engineering**

Ranked important attributes using Information Gain
Identified the strongest predictive features
Removed non-informative variables

**✔ Machine Learning Models**

Implemented and evaluated:

J48 Decision Tree
Naïve Bayes
Random Forest
K-Means Clustering

**✔ Model Evaluation**

Compared model performance using:

Accuracy
Precision
Recall
F-Measure
ROC Curve
